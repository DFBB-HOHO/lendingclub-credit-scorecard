# 简历项目条目（风险策略 / 量化建模岗）

## LendingClub 贷前信用评分卡｜个人项目

**Python · WOE/IV · Logistic Regression · AUC/KS · PSI · Credit Policy**

- 基于 LendingClub 2007–2018 公开信贷数据构建贷前申请评分卡；按放款时间划分开发集（2014–2015，59.9 万笔）、验证集（2016，29.3 万笔）与 OOT 测试集（2017，16.9 万笔），并识别 2018 年样本的表现期右删失风险。
- 建立字段泄露清单，排除 `grade`、`sub_grade`、`int_rate` 及贷后还款字段；完成缺失处理、训练期缩尾、单调 WOE 分箱、IV/相关性筛选，保留 15 个贷前申请与征信变量构建 L2 Logistic 评分卡。
- OOT AUC 0.681、KS 26.2%，最高风险十分位坏账率 45.2%、最低风险十分位 6.1%；完成 PDO50 分数映射、验证期 PD 截距校准与 PSI 监控（Score PSI 0.023）。
- 设计通过率 30%–90% 的策略阈值情景并明确批准样本偏差、拒绝推断与跨市场迁移局限；沉淀可复现代码、单元测试、模型开发报告及模型治理材料。

> 面试口径：本项目的重点不是“把 AUC 做高”，而是证明我能把时间切分、标签成熟度、字段泄露、校准、稳定性和策略阈值放进同一套贷前风险建模闭环。策略情景是历史批准样本上的描述性模拟，不等同于真实全量申请通过率。
